ServiceListenerInterface

Extends \Zend\EventManager\ListenerAggregateInterface

Interface for self-registering event listeners.

Classes implementing this interface may be registered by name or instance with an EventManager, without an event name. The \Zend\ModuleManager\Listener\attach() method will then be called with the current EventManager instance, allowing the class to wire up one or more listeners.

addServiceManager(string $serviceManager, string $key, string $moduleInterface, string $method)
  • $serviceManager is the service name for the service/plugin manager.
  • $key is the configuration key containing configuration for it.
  • $moduleInterface is the interface indicating a configuration provider for it.
  • $method is used for duck-typing configuration providers.
