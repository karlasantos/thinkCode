SingleTablePersister

Extends \Doctrine\ORM\Persisters\Entity\AbstractEntityInheritancePersister

Persister for entities that participate in a hierarchy mapped with the SINGLE_TABLE strategy.

All these persisters are assumed to use a discriminator column to discriminate entity types in the hierarchy.
