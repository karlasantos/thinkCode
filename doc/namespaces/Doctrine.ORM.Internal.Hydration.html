IterableResult¶

Represents a result structure that can be iterated over, hydrating row-by-row during the iteration. An IterableResult is obtained by AbstractHydrator#iterate().

ObjectHydrator¶

The ObjectHydrator constructs an object graph out of an SQL result set.

ScalarHydrator¶

Hydrator that produces flat, rectangular results of scalar data.

The created result is almost the same as a regular SQL result set, except that column names are mapped to field names and data type conversions take place.
